## `POST /wallet/portal`

Returns a URL to Stripe's hosted Billing Portal, where users manage saved payment methods and view invoices.

### Request

```json
{ "return_url": "https://example.com/wallet" }
```

<ParamField body="return_url" type="string">
  URL Stripe returns the user to when they leave the portal. Optional.
</ParamField>

### Response

```json
{ "success": true, "data": { "portal_url": "https://billing.stripe.com/p/session/..." } }
```

<Warning>
  **The `portal_url` is single-use and short-lived.** Always request a fresh one immediately before redirecting the browser. Never cache it or embed it in emails.
</Warning>
